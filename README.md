##System:
    + OS: Mac OS x 10.7.5
    + Python 3.3 PyQt4

##How to play:
    + `python3 fishy.py`
    + use arrow keys to control the "hunter" (red dot) to chase "fish" (blue triangle). When all fish are caught a congratulation message will display.
    + press q to quit at any time.
